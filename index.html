YouTube 좋아요 영상 필터

채널 및 키워드로 좋아요한 YouTube 영상 찾기

크롬 확장 프로그램 개요

이 확장 프로그램은 YouTube에서 좋아요한 영상을 쉽게 찾을 수 있도록 도와줍니다. 채널명이나 키워드로 필터링할 수 있어 원하는 영상을 빠르게 찾을 수 있습니다.

주요 기능

  • 구글 계정으로 로그인하여 YouTube 좋아요 영상 접근
  • 채널명으로 좋아요한 영상 필터링
  • 제목 또는 설명에 포함된 키워드로 검색
  • 깔끔한 UI로 영상 목록 표시
  • 확장 프로그램 설정 페이지에서 다양한 옵션 설정

설치 및 사용법

이 크롬 확장 프로그램을 사용하려면 다음 단계를 따르세요:

  1. Google 클라이언트 ID 얻기
    Google 개발자 콘솔에서 새 프로젝트를 만들고 OAuth 클라이언트 ID를 생성하세요. YouTube Data API를 활성화하고 OAuth 동의 화면을 설정해야 합니다.
  2. 클라이언트 ID 설정
    manifest.json 파일에서 ${GOOGLE_CLIENT_ID} 부분을 실제 클라이언트 ID로 변경하세요.
  3. 크롬에 확장 프로그램 로드
    Chrome 브라우저에서 chrome://extensions로 이동한 후, 개발자 모드를 활성화하고 "압축해제된 확장 프로그램 로드" 버튼을 클릭하여 이 폴더를 선택하세요.
  4. 확장 프로그램 사용
    브라우저 상단의 확장 프로그램 아이콘을 클릭하여 "YouTube 좋아요 영상 필터"를 사용하세요. 구글 계정으로 로그인한 후 좋아요한 영상을 볼 수 있습니다.

소스 코드 구조

이 확장 프로그램은 다음과 같은 주요 파일로 구성되어 있습니다:

  • manifest.json - 확장 프로그램 설정 및 권한
  • popup.html/js/css - 메인 팝업 인터페이스
  • background.js - 인증 및 백그라운드 작업 관리
  • options.html/js/css - 설정 페이지
  • utils.js - 공통 유틸리티 함수
  • content.js - YouTube 페이지와의 상호작용

API 사용

이 확장 프로그램은 다음 API를 사용합니다:

  • YouTube Data API v3 - 사용자의 좋아요한 영상 가져오기
  • Google OAuth 2.0 - 사용자 인증
  • Chrome Extension API - 스토리지, 메시지 전달, 인증 등

개발 참고 사항

이 확장 프로그램을 수정하거나 개발할 때 다음 사항을 참고하세요:

  • JavaScript 모듈 구조로 설계되어 있으며, 각 파일은 특정 기능을 담당합니다.
  • Chrome의 local storage를 활용하여 데이터를 캐싱합니다.
  • YouTube Data API v3 할당량에 주의하세요. 과도한 API 호출은 할당량 제한에 도달할 수 있습니다.
  • 사용자 인증 상태는 Chrome의 identity API를 통해 관리됩니다.
// API 호출 예시
fetch('https://www.googleapis.com/youtube/v3/videos?part=snippet,contentDetails,statistics&myRating=like&maxResults=50', {
  headers: {
    'Authorization': `Bearer ${accessToken}`
  }
})
